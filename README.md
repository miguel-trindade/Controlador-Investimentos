# 💰 Controlador de Investimentos

Aplicação desenvolvida em Python para auxiliar usuários no planejamento financeiro e na simulação de investimentos.

## 📖 Sobre o Projeto

O Controlador de Investimentos é um sistema executado via terminal que permite ao usuário realizar simulações financeiras e estimar o tempo necessário para atingir metas de investimento.

O objetivo é fornecer uma ferramenta simples e prática para apoiar o planejamento financeiro pessoal.

## ✨ Funcionalidades

- 📈 Simulação de investimentos
- 🎯 Cálculo de tempo para atingir uma meta financeira
- 💵 Planejamento financeiro básico
- 🖥️ Interface via terminal
- 🔢 Cálculos automáticos de rentabilidade

## 🛠️ Tecnologias Utilizadas

- Python 3

## 🚀 Como Executar o Projeto

### Pré-requisitos

- Python 3 instalado na máquina

### Passos

1. Clone o repositório:

```bash
git clone https://github.com/miguel-trindade/Controlador-Investimentos.git
```

2. Acesse a pasta do projeto:

```bash
cd Controlador-Investimentos
```

3. Execute o arquivo principal:

```bash
python financas.py
```

## 📋 Exemplos de Uso

### Simulação de Investimento

O usuário informa:

- Valor inicial
- Valor dos aportes mensais
- Taxa de rendimento
- Tempo de investimento

O sistema calcula o valor acumulado ao final do período.

### Meta Financeira

O usuário informa:

- Valor da meta
- Investimento inicial
- Aporte mensal
- Rentabilidade esperada

O sistema estima o tempo necessário para atingir o objetivo.
